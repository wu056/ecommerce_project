# 电商用户行为分析项目

这是一个使用Python Pandas对电商销售数据进行深入分析的项目。

## 项目内容

-   **数据清洗**：处理缺失值、重复值，转换日期格式。
-   **复购分析**：计算整体复购率，分析不同品类的复购情况。
-   **高价值订单分析**：筛选出金额大于500且已交付的订单，分析其地域分布。
-   **物流分析**：通过数据透视表分析不同物流服务等级的表现。

## 技术栈

-   Python
-   Pandas
-   NumPy

## 如何运行

1.  确保安装了Python和必要的库：
    ```bash
    pip install pandas numpy
    ```
2.  克隆本项目到本地：
    ```bash
    git clone https://github.com/你的用户名/ecommerce-project.git
    ```
3.  运行主分析脚本：
    ```bash
    python 你的论文名字.py
    ```

## 核心发现

-   发现了复购率极高的核心品类（如Set、kurta）。
-   识别出高价值订单集中的重点城市（如孟买、班加罗尔）。
-   分析了物流服务质量对订单状态的影响。
